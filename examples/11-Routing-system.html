Have a look at paulajs.com website source code for a live sample https://paulajs.com/routing

1) Define routes

<jpau-content>
  <jpau-route path="/" module="index"></jpau-route>
  <jpau-route path="/mypage" module="mypage"></jpau-route>
  <jpau-route path="(default)" module="404"></jpau-route>
</jpau-content>
- In this example / for index module
- mypage.js for an example
- 404.js for (default) if location does not match any route


2) Create links (with is="jpau-link")

<a href="/" is="jpau-link">Home</a>
<a href="/mypage" is="jpau-link">My Page</a>

3) Create (example) module mypage.js

export const data = {
	title: 'Page title',
	myvar: 'My variable'
};

export const funcs = {
	onLoad: () => data.myvar = 1,
	onMount: function() {},
	myFunction: function(i) {this.data.myvar = i}
};

export const html = \`
<h1>Page content</h1>
\`;
